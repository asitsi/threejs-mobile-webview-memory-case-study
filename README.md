# Memory Management in Three.js Mobile WebViews

This repository contains a technical case study about improving memory
management in a Three.js and React Three Fiber driving application running
inside Android and iOS WebViews.

## Document

[Read the technical case study](paper/threejs-mobile-webview-memory-case-study.pdf)

## Topics covered

- Sliding-window route streaming
- Three.js resource disposal
- React Three Fiber performance
- Asset caching and ownership
- Mobile WebView memory constraints
- Predictive chunk loading
